# React-Admin-Dashboard

This is a personal project built to display admin dashboard analytics. It is developed using React and Tailwind CSS.

## Features

### The Admin dashboard includes:

- Sidebar - Contains the nav links for each pages and collapsable for smaller screens.
- Topbar - Includes the notification and profile. Both has a dropdown/modal to expand additional information.

### Pages

- Home - Displays summary cards, charts for monthly and weekly sales, category breakdowns, and recent user activity.
- Analytics - Separate views of the data.
- Users – Displays a list of users with full CRUD (Create, Read, Update, Delete) functionality.

## Charts

Charts are implemented using Chart.js to visualize data in various formats.
